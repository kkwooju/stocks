주식 포트폴리오 차트는 보유 종목을 등록하면 실시간 시세와 환율로 평가금액·수익률·비중을 자동 계산해 주는 무료 웹앱입니다. 회원가입 없이, 데이터는 내 기기에만 저장됩니다.

핵심 기능

여러 나라에 흩어진 종목을, 환율까지 반영해 하나의 그림으로.

🌏 다국가 시장

한국·미국·일본·홍콩·대만 등 12개 시장 지원. 시장을 고르면 통화·환율이 자동 반영됩니다.

💱 시세·환율 자동

Yahoo Finance에서 현재가와 KRW 대비 환율을 받아 평가금액·수익률을 계산(약 15분 지연).

🍩 비중 도넛 차트

평가금액 기준과 매수금액 기준 두 가지로 비중을 시각화해 리밸런싱 신호를 확인.

📈 수익률 계산

매수단가 대비 현재가 수익률을 종목별·전체로 표시. 같은 종목 여러 계좌는 가중평균으로 합산.

📱 PWA 설치

홈 화면에 추가하면 앱처럼 사용. 갤럭시·아이폰·PC 모두 앱스토어 없이 즉시 설치.

🔒 로컬 저장

보유 종목·매수단가·수량은 내 기기 localStorage에만. 서버 전송·광고·추적 없음.

4단계로 시작하기

5분이면 본인 포트폴리오를 등록하고 수익률·비중을 볼 수 있어요.

  1. 시연 종목 확인접속하면 한국·미국 ETF 5개 시연 데이터가 보입니다. 동작을 먼저 확인하세요.
  2. + 종목 추가티커(시장 코드 제외 본체, 예: AAPL·069500·2330)와 시장·수량·매수단가를 입력합니다.
  3. 자동 계산저장하면 현재가·회사명·환율을 받아 수익률과 평가금액을 자동 계산합니다.
  4. 시연 종목 정리시연 종목을 삭제하면 나만의 포트폴리오 완성. 이후 변경은 자동 저장됩니다.

자주 묻는 질문

데이터 보관, 지원 시장, 가격 갱신에 대한 답변.

내 보유 종목 데이터는 어디에 저장되나요?
보유 종목·매수단가·수량은 사용 중인 기기의 브라우저 localStorage에만 저장됩니다. 서버나 클라우드로 전송되지 않으며 광고·트래커도 없습니다. 가격·환율을 받아올 때 Yahoo Finance에 티커 심볼만 전달되고 매수단가·수량 등 개인 정보는 외부로 나가지 않습니다.
어떤 시장(국가)의 주식을 등록할 수 있나요?
한국(코스피·코스닥), 미국, 일본, 홍콩, 대만, 네덜란드, 영국, 프랑스, 독일, 이탈리아, 중국, 스위스 등 12개 시장을 지원합니다. 종목 추가 시 시장을 선택하면 통화와 티커 접미사가 자동 결정되고, 외화 종목은 원화(KRW) 환율로 환산해 평가합니다.
가격이 갱신되지 않을 때는 어떻게 하나요?
대부분 시장 선택 오류입니다. 예를 들어 네덜란드 종목 BESI를 미국으로 설정하면 조회되지 않으니 올바른 시장으로 변경하세요. 공개 프록시의 일시 장애일 수도 있으니 잠시 후 '가격 갱신'을 다시 누르면 됩니다. 시세는 약 15분 지연됩니다.
다른 사람에게 내 포트폴리오가 보이나요?
아니요. localStorage는 해당 기기의 해당 브라우저에만 격리됩니다. 같은 주소를 다른 사람이 열어도 시연용 데이터만 보이고 본인 데이터는 노출되지 않습니다.
수익률 색상이 미국 주식 사이트와 반대인 이유는?
한국식 관례를 따라 상승은 빨강, 하락은 파랑으로 표시합니다. 미국식(상승 초록·하락 빨강)과 반대입니다.
회원가입이나 비용이 필요한가요?
필요 없습니다. 회원가입·로그인·결제 없이 무료로 사용합니다. 광고와 추적도 없으며 소스 코드 전체가 GitHub에 공개돼 있습니다.
