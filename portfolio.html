주식 포트폴리오 사용 가이드

5분이면 시작할 수 있어요. 본인 데이터는 디바이스 안에만 안전하게 보관됩니다.

내 종목 데이터는 어디로 가나요?

아무 데도 안 갑니다. 본인의 보유 종목·매수단가·수량은 이 디바이스의 localStorage에만 저장됩니다. 서버·클라우드로 전송되지 않으며, 광고·트래커도 없어요.

가격·환율을 받아오기 위해 Yahoo Finance API만 호출합니다 — 이때 티커 심볼만 전달되고 매수단가·수량 등은 외부에 나가지 않습니다.

🚀 4단계로 시작하기

1
홈 화면에서 시연 종목 확인
처음 접속 시 한국·미국 ETF 5개 시연 데이터가 보입니다. 본인 데이터로 교체하기 전에 동작을 확인.
2
+ 종목 추가 클릭 → 본인 종목 입력
티커는 시장 코드를 뺀 본체만 — 예: AAPL, 069500, 2330(대만 TSMC). 시장 드롭다운에서 거래소(🇰🇷 코스피, 🇺🇸 미국, 🇹🇼 대만 등)를 선택하면 통화·접미사 자동 결정.
3
자동으로 현재가·회사명·환율 갱신
저장 즉시 Yahoo Finance에서 현재가·회사명·KRW 대비 환율을 가져와 수익률·평가금액을 계산합니다. ⓘ 15분 지연 시세이므로 실시간 매매용은 아닙니다.
4
시연 종목 삭제 + 자신만의 포트폴리오 완성
데스크탑은 행의 ✕ 버튼, 모바일은 행을 탭해 편집 모달의 🗑 삭제로 시연 종목 정리. 이후 변경 사항은 자동 저장됩니다 — 페이지를 닫았다 다시 열어도 그대로 유지.

💾 데이터 백업 · PC·모바일 동기화

디바이스 안에만 저장되므로 PC와 모바일은 각자 별개로 데이터가 쌓입니다. 두 기기에서 같은 종목을 보려면 한 쪽에서 저장 → 다른 쪽에서 불러오기.

💾 파일로 저장
홈 화면의 💾 저장 버튼 → portfolio_data.json 다운로드
📂 파일에서 불러오기
홈 화면의 📂 불러오기 → 받은 JSON 파일 선택 → 즉시 적용
🔗 파일 연결 (PC만)
Chrome·Edge에서 한 번 연결 → 이후 변경 시 자동 저장 (다운로드 폴더 안 거침)
🗑 초기화
시연 데이터로 되돌리기 — 본인 데이터는 사라지므로 백업 후 실행
💡 동기화 팁: PC에서 💾 저장 → 카톡으로 본인에게 전송 → 모바일에서 📂 불러오기. 반대 방향도 동일. 자동 sync는 아니지만 가장 안전하고 빠른 방법입니다.

📱 앱처럼 홈 화면에 설치하기

웹사이트를 모바일 앱처럼 사용할 수 있습니다 (PWA). 별도 앱스토어 등록 없이 즉시 가능.

A
갤럭시·안드로이드 Chrome
우상단 📱 앱 설치 버튼 클릭 → 자동 설치. 또는 메뉴 ⋮ → "홈 화면에 추가".
B
아이폰·아이패드 Safari
하단 공유 버튼 □↑ → "홈 화면에 추가". 도넛+₩ 아이콘이 홈에 생성됩니다.
C
PC Chrome·Edge
주소창 우측 설치 아이콘 ⊕ 클릭 → 별도 창으로 앱처럼 실행.

🎯 핵심 기능

  • 다국가 시장 지원: 한국·미국·일본·홍콩·대만·네덜란드·영국·프랑스·독일·이탈리아·중국·스위스 — 시장 드롭다운에서 선택
  • 도넛 차트 두 기준: 평가금액 기준(현재 비중) vs 매수금액 기준(초기 투입) — 리밸런싱 신호 파악
  • 한국식 색상: 🔴 상승 / 🔵 하락 (미국식과 반대)
  • 도넛 조각 호버·탭: 작은 조각은 라벨이 안 보여도 호버하면 종목명·평가금액·수익률 툴팁 표시
  • 레전드 정렬: 평가금액·수익률·종목명·비율 4가지 기준 + 오름차순/내림차순 토글
  • 표 헤더 정렬: 데스크탑에서 "티커"·"수익률" 클릭 시 정렬 (모바일은 카드 리스트라 정렬 없음)

❓ 자주 묻는 질문

Q. 가격이 안 올라와요

거의 항상 시장 선택 오류입니다. 예: BESI를 🇺🇸 미국으로 두면 안 됨 → 🇳🇱 네덜란드로 변경. 공개 CORS 프록시 일시 장애일 수도 있으니 잠시 후 🔄 가격 갱신 재시도.

Q. 데이터가 사라졌어요

브라우저 시크릿 모드는 종료 시 localStorage가 삭제됩니다. 일반 모드 사용 권장. 또 브라우저 "사이트 데이터 삭제"하면 사라지므로 정기적으로 💾 백업.

Q. 다른 사람에게 내 데이터가 보이나요?

아니요. localStorage는 해당 디바이스의 해당 브라우저에만 격리됩니다. 같은 URL을 다른 사람이 열어도 시연 데이터만 보입니다.

Q. 수익률 색이 미국 주식 사이트와 반대예요

한국식 관례를 따라 🔴 상승 / 🔵 하락입니다.
